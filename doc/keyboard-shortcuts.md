# Keyboard Shortcuts

Quick reference for all keyboard shortcuts in Mauve Viewer. Press **?** to toggle the shortcuts help panel inside the viewer.

## Navigation

| Shortcut | Action |
|----------|--------|
| Ctrl + Up | Zoom in 2x |
| Ctrl + Down | Zoom out 2x |
| Ctrl + Scroll wheel | Zoom at mouse position |
| Ctrl + Left | Pan left 10% |
| Ctrl + Right | Pan right 10% |
| Ctrl + Shift + Left | Pan left 20% |
| Ctrl + Shift + Right | Pan right 20% |

## Selection

| Shortcut | Action |
|----------|--------|
| Shift + Drag | Select a region |
| Shift + Click | Clear selection (click without dragging) |
| Escape | Clear selection or close dialog |

## Tools

| Shortcut | Action |
|----------|--------|
| Ctrl + I | Open Sequence Navigator |
| Ctrl + E | Export image |
| Ctrl + P | Print |
| ? | Toggle keyboard shortcuts help |
